# Upstream reading map (open sources)

Use these sections as a quick “where do I look next” guide:

## Command Line Applications in Rust (rust-cli book)

- Getting started: https://rust-cli.github.io/book/index.html
- 15-minute tutorial:
  - Project setup: https://rust-cli.github.io/book/tutorial/setup.html
  - Parsing CLI args: https://rust-cli.github.io/book/tutorial/cli-args.html
  - Nicer errors: https://rust-cli.github.io/book/tutorial/errors.html
  - Output: https://rust-cli.github.io/book/tutorial/output.html
  - Testing: https://rust-cli.github.io/book/tutorial/testing.html
  - Packaging: https://rust-cli.github.io/book/tutorial/packaging.html
- In-depth topics:
  - Signals: https://rust-cli.github.io/book/in-depth/signals.html
  - Config files: https://rust-cli.github.io/book/in-depth/config-files.html
  - Exit codes: https://rust-cli.github.io/book/in-depth/exit-code.html
  - Docs: https://rust-cli.github.io/book/in-depth/docs.html

## Books (closed sources)

- “Command-Line Rust” (Ken Youens-Clark): project-based CLI patterns (tests, IO, composability).
- “The Secrets of Rust: Tools”: Rust tooling and test habits (cargo, clippy, iteration).
